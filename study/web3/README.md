# 什么是 Web3

Web3（又叫做 Web3.0 ）是建立在区块链技术之上的下一代互联网。它强调去中心化，即没有中心化的控制机构来管理数据和资源。在 Web3 的世界里，用户对自己创造的内容和数据拥有真正的所有权，从而摆脱对传统中心化平台的依赖。此外，通过智能合约和区块链技术实现的不可篡改和透明性特征，可以促进信任的建立，降低欺诈和中介成本。Web3 还激发了新的商业模式和经济系统，如去中心化金融（ DeFi ）、非同质化代币（ NFTs ）和去中心化自治组织（ DAOs ）等等。

### [Solana](https://www.theblock.co/learn/249535/what-is-solana-and-how-does-it-work-a-beginners-guide-to-sol-cryptocurrency)

Solana 是一个区块链平台，旨在托管去中心化、可扩展的应用程序。它成立于 2017 年，是一个开源项目，目前由位于日内瓦的 Solana 基金会运营，而区块链则由总部位于旧金山的 Solana Labs 构建。它提供了快速、便宜且可扩展的交易体验，支持使用Rust，C++和C语言来编写智能合约。

Solana 的主要显着特征之一是其权益证明 （PoS） 共识系统，该系统称为 Tower Consensus 或 Tower BFT。它使分布式网络能够达成共识，尽管来自恶意节点的攻击，称为拜占庭容错 （PBFT）。
