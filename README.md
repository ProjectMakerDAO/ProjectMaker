# ProjectMaker
ProjectMakerDAO社群旨在以项目化的方式让大家真正进入Web3领域，建设目标：

1. 项目化学习，以项目为导向，持续项目开发与内部孵化
2. 共享资源，共享各种Web3技术、应用、行业等资讯
3. 共建社群，社区最终以DAO的形式存在，全体成员共同建设管理社群
4. 职业共助，社区分享工作机会、相关经验等
5. Study To Earn，社群每个人的贡献会以链上Token的形式记录
